What Is Fandom Trivia?

Meet the app — a trivia battleground where fandoms collide.

A Trivia App for Superfans

Fandom Trivia is a single-page web app where you test your knowledge of your favorite universes — Twilight, Harry Potter, K-Pop, The Three-Body Problem, and Zootopia. Pick a fandom, answer multiple-choice questions, earn badges, and climb the leaderboard.

It's built with modern web tech: React 19 + TypeScript for the UI, Supabase for the database and login system, and Tailwind CSS for styling. Everything runs in your browser — no installs needed.

🧠 Why this matters: Even though this is "just a quiz app," it uses the same architecture as production web apps used by millions. Authentication, databases, UI state management, animations — every concept here scales to real-world products.

Five Worlds, One App

Each fandom is a "universe" — a bundle of questions, a theme, and a personality. The app treats them all the same way internally, which is why adding a new fandom is just a matter of adding data.

🧛

Twilight

30 questions about Forks, Cullens, and Quileute legends. Volume I chapters 1–8.

⚡

Harry Potter

Wizarding World questions spanning Sorcerer's Stone through Deathly Hallows.

🎤

13 questions about the Saja Boys, Rumi, Jinu, and the Huntr/x agency.

🌌

Three-Body Problem

Sci-fi questions about the Dark Forest, Trisolaran crisis, and higher dimensions.

🐰

Zootopia

Urban jungle mysteries with Judy Hopps, Nick Wilde, and conspiracy cases.

What a Universe Looks Like in Code

Each universe is just a JavaScript object in a file called constants.ts. Here's the actual code for the Twilight universe — every field powers what you see on screen.

CODE 

{
  id: 'twilight',
  title: 'Twilight',
  tags: ['Fantasy', 'Mystery'],
  description: 'Misty forests & eternal secrets.',
  buttonText: 'Enter Forks',
  icon: 'Droplets',
  image: '/images/Cullen Family.jpg'
}
PLAIN ENGLISH

A "universe" object starts here.

Internal ID — used to look up questions for this fandom.

The big title you see on the card.

Tags are like genre labels (Fantasy, Mystery) shown on the landing page.

The short tagline that appears below the title.

The text on the button ("Enter Forks") — it's different for each universe!

Lucide icon name — tells the app which icon to draw.

Background image path for the card.

Close of the object.

💡 Key insight: Because each universe is just data in an array, adding "Star Wars" or "Marvel" only means adding one more object — zero new code required. That's the power of data-driven design.

More Than Just Questions

The app has four main areas. Together they turn a simple quiz into a full gamified experience.

📝

Quiz

Multiple-choice questions with instant feedback, answer evidence, and a progress tracker. The heart of the app.

🏆

Rankings

Global leaderboard showing top scorers across all universes. Pulled live from Supabase.

⭐

Badges

Unlockable achievements like "Forks Resident" (complete a Twilight quiz) or "Perfectionist" (100% on any quiz).

📊

Dashboard

"Pro Hub" with live leaderboard, active tournaments, and stats. The competitive layer.
